# 为防止熊孩子上网太晚，偷偷搞一个 crontab
* crontab -e
```
# 每天 23:10 打开代理
10 23 * * * /usr/sbin/networksetup -setwebproxy "Wi-Fi" 127.0.0.1 3721 && /usr/sbin/networksetup -setsecurewebproxy "Wi-Fi" 127.0.0.1 3721
# 每天 07:00 清除代理
0 7 * * * /usr/sbin/networksetup -setwebproxy "Wi-Fi" "" 0 && /usr/sbin/networksetup -setsecurewebproxy "Wi-Fi" "" 0
```

* crontab -l # 查看定时任务

* 命令使用方法：
```
# 手动测试 - 启用代理
networksetup -setwebproxy Wi-Fi 127.0.0.1 3721
networksetup -setsecurewebproxy Wi-Fi 127.0.0.1 3721
# 验证设置
networksetup -getwebproxy Wi-Fi
networksetup -getsecurewebproxy Wi-Fi
# 恢复默认设置
networksetup -setwebproxystate Wi-Fi off
networksetup -setsecurewebproxystate Wi-Fi off
```

* 一键消除：
```
sudo networksetup -setwebproxy "Wi-Fi" "" 0 && sudo networksetup -setsecurewebproxy "Wi-Fi" "" 0
```
