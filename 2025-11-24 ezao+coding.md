# 先调一个网友发的claude zelda 声音插件
* https://www.vibesparking.com/zh-cn/blog/ai/claude-code/zelda-claude-code/2025-08-18-zelda-sounds-for-claude-code/
    zelda-claude-code 是一个给 Claude Code 加音效与成就系统的 npm 包：
        - 成功/失败、连击、成就、会话开始/结束……都有不同音效；
        - 内置 统计面板：成功率、连胜、最强连击、最常用工具；
        - 直接在 Claude Code 里用 @zelda 系列命令调音量、开关提示、看数据
        - 帮助 @zelda help
* npm install -g zelda-claude-code@latest 之后重新进入claude
